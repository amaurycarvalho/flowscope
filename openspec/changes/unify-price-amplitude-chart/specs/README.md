# Specs — Unify Price Amplitude Chart

Nenhuma capability nova ou modificada. Esta mudança é puramente de implementação da UI — reorganização visual do painel "Amplitude de Preço" sem alteração de requisitos funcionais.

## Justificativa

Conforme definido no proposal.md: a fusão dos três sub-gráficos (Price Range Timeline, Range % Histórico, Eficiência Diária) num único axes não altera o que o sistema faz, apenas como apresenta visualmente. Portanto, não há specs a criar ou modificar.
